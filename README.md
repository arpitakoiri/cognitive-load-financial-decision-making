# Cognitive Load and Financial Decision-Making

## Overview

This project investigates whether cognitive load influences financial decision-making and risk-taking behaviour. A Streamlit-based behavioural experiment was developed in Python to examine how individuals make financial choices under varying levels of mental load.

## Research Question

Does cognitive load influence financial risk-taking and reaction time during financial decision-making tasks?

## Methodology

* Participants were assigned either a low-load or high-load memory task.
* Participants completed multiple financial decision-making trials.
* Each trial involved choosing between a guaranteed reward and a probabilistic reward.
* Reaction times were automatically recorded.
* Experimental data was stored in CSV format for analysis.

## Tools Used

* Python
* Streamlit
* Pandas
* NumPy
* SciPy
* Matplotlib

## Analysis

The collected data was analyzed using descriptive statistics, visualizations, and independent-samples t-tests to compare reaction times across cognitive load conditions.

## Results

Pilot analysis found no statistically significant difference in reaction times between high-load and low-load conditions (t = 0.90, p = 0.376). Additional participants and trials would be required for stronger conclusions.

## Future Improvements
